# portfolio-growth-viz
Portfolio growth visualizer. Written by ChatGPT5.3 in ~30 min. No human code.

## Features:

* Add/remove stocks
* Load/save portfolio as .csv
* Compare various portfolios (by loading with `Compare`) or stocks (checkbox)
* Various time ranges
* Scrollable portfolio if it's too long
